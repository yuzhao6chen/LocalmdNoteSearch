# Rust 大作业实验报告要点

## 项目目标

报告需要说明项目要解决什么问题。本项目目标是实现一个本地 Markdown 知识库搜索系统，
帮助用户快速查找课程笔记和技术文档。

## 模块设计

可以重点介绍 ingest、index、search 和 app 四个模块。每个模块都有清晰职责，便于维护
和测试。

## 测试与工程规范

报告中应展示 cargo fmt、cargo clippy 和 cargo test 的结果，并说明关键单元测试覆盖了
解析、索引、排序、缓存和 TUI 交互。
